The Mandelbrot and Julia Sets!

The fractal shown will alternate between the Mandelbrot Set and a Julia Set generated based on where on the Mandelbrot Set you click.

Coordinates: The Mandelbrot Set is on a complex coordinate plane. The y-coordinates go from -1.5i to 1.5i and the x-coordinates go from -2 to 2. The Julia Set is generated from the complex number that corresponds to your mouse coordinates.

Zooming: To zoom in or zoom out on the fractal, use the keyboard. The up and right arrow keys and the letter 'z' will zoom in on the fractal, while the down and left and letter 'x' will zoom out from the fractal.

You can download an offline application for here for macs, Windows 32, and Windows 64. You can download an offline application This will run faster and has more options than the online version.
