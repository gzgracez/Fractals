The Mandelbrot and Julia Sets!

The fractal shown will alternate between the Mandelbrot Set and a Julia Set generated based on where on the Mandelbrot Set you click.

The Mandelbrot Set is on a complex coordinate plane. The y-coordinates go from -1.5i to 1.5i and the x-coordinates go from -2 to 2. The Julia Set is generated from the complex number that corresponds to your mouse coordinates.
